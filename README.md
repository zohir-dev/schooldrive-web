
# Driving School Website

Driving school website that displays all available driving licenses, organized by type (manual, automatic, and adapted). It allows new students to register directly from the website.
## Tech Stack

 HTML5 & CSS3

## 🚀 Live Preview
You don't need to download the code to see the project in action. You can check out the live website here:
**[View Live Demo](https://htmlpreview.github.io/?https://github.com/zohir-dev/schooldrive-web/blob/main/HTML%20_CSS/index.html)**

## Usage/Examples

Clone the repository and open `index.html` in any browser. No installation required.

## Features

- Licence catalogue organised by type: B, A, A1, A2, AM, C and D
- Variants available per licence: manual, automatic and adapted
- Individual page for each licence with specific information
- Student enrolment form with personal details and licence selection
- Custom stylesheet for each section of the site
